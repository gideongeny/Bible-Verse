## ğŸ› ï¸ Tech Stack & Architecture

This project utilizes the latest Android [Jetpack](https://developer.android.com/jetpack) components and follows modern development best practices.

-   **Language:** [Kotlin](https://kotlinlang.org/)
-   **Architecture:** [MVVM](https://developer.android.com/jetpack/guide) (Model-View-ViewModel)
-   **Dependency Injection:** [Dagger-Hilt](https://developer.android.com/training/dependency-injection/hilt-android)
-   **Local Database:** [Room](https://developer.android.com/jetpack/androidx/releases/room)
-   **Networking:** [Retrofit](https://square.github.io/retrofit/) & [Gson](https://github.com/google/gson)
-   **Navigation:** [Navigation Component](https://developer.android.com/guide/navigation)
-   **Image Loading:** [Glide](https://github.com/bumptech/glide)
-   **UI:** [ViewBinding](https://developer.android.com/topic/libraries/view-binding), Material Components
